# Plaid Teardown

> ⚠️ Draft — needs verification against primary sources

## Summary
An analysis of Plaid, the data aggregator that became the de facto infrastructure for connecting fintech apps to consumer bank accounts in North America.

## Key Concepts
- **Screen Scraping to APIs**: How Plaid moved from harvesting data via user credentials to formal API agreements with major banks.
- **The "OAuth" Shift**: The transition toward token-based authentication to improve security and reliability.
- **Product Expansion**: Moving beyond data aggregation into payment initiation, identity verification, and credit risk assessment.
- **The Visa Acquisition Attempt**: Lessons from the DOJ's block of the $5.3B acquisition in 2021.
