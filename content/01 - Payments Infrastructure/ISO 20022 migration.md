# ISO 20022 migration

> ⚠️ Draft — needs verification against primary sources

## Summary
The global movement toward a unified, XML-based data standard for financial messaging, replacing legacy MT and BAI formats.

## Key Concepts
- **Rich Data**: How ISO 20022 allows for more detailed information (sender/receiver addresses, purpose of payment) to be attached to a transaction.
- **Interoperability**: Standardizing messaging between SWIFT, FedNow, TCH, and other global payment rails.
- **Regulatory Reporting**: Reducing friction in AML/KYC screening by providing structured data that is easier for machines to parse.
- **Migration Timeline**: The multi-year transition period for financial institutions to upgrade their core banking systems and middleware.
