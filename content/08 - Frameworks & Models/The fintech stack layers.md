# The fintech stack layers

> ⚠️ Draft — needs verification against primary sources

## Summary
The vertical decomposition of the hardware, software, and regulatory layers that make up a modern financial service.

## Key Concepts
- **The Capital Layer**: The underlying source of funds (the bank charter or the credit line).
- **The Core Layer**: The system of record (the ledger) which manages accounts and balances.
- **The Connectivity Layer**: The APIs and middleware that connect the core to the outside world.
- **The Application Layer**: The consumer-facing UI and logic where the end-user interacts with the service.
