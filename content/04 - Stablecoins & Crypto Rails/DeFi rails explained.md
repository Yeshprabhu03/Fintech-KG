# DeFi rails explained

> ⚠️ Draft — needs verification against primary sources

## Summary
The peer-to-peer financial system built using smart contracts on programmable blockchains, removing traditional intermediaries like banks and brokers.

## Key Concepts
- **Automated Market Makers (AMMs)**: Protocols (e.g., Uniswap) that use mathematical formulas to price assets and allow for decentralized trading.
- **Liquidity Pools**: Crowdsourced pools of tokens that provide the liquidity needed for decentralized exchanges.
- **Composable Finance**: The "Money Lego" concept where different DeFi protocols can be stacked to create complex financial products.
- **Over-Collateralized Lending**: Borrowing assets by providing more than 100% of the loan value in another token as collateral (e.g., Aave, Compound).
